# Intrusion-Detection-System-Machine-Learning-

-Project Description

A machine learning-based intrusion detection system designed to classify and detect different types of cyberattacks using network traffic data.

-Key Features

Detection of attack types such as:
Mirai GRE Flood
Recon OS Scan
Dictionary Brute Force
Feature engineering on network traffic data
Model training and evaluation using multiple classifiers
Performance comparison between ML models

-Technologies Used

Python, Scikit-learn, Pandas, NumPy, XGBoost, Random Forest, Jupyter Notebook

-Outcome

Improved detection accuracy of malicious network activity and demonstrated the effectiveness of ML in cybersecurity threat detection.
